## Тест-данные
- Valid user: `standard_user`
- Password: `secret_sauce`
- Locked user: `locked_out_user`

## Логин

**TC-LOGIN-001 — Успешный вход (валидные данные)**
- Предусловия: открыта страница https://www.saucedemo.com
- Шаги:
  1) Ввести username: `standard_user`
  2) Ввести password: `secret_sauce`
  3) Нажать `Login`
- Ожидаемый результат: открылась страница каталога товаров (Products), отображается список товаров.

**TC-LOGIN-002 — Ошибка при неверном пароле**
- Предусловия: открыта страница логина
- Шаги:
  1) Ввести username: `standard_user`
  2) Ввести password: `wrong_password`
  3) Нажать `Login`
- Ожидаемый результат: вход не выполнен, показано сообщение об ошибке (error message).

**TC-LOGIN-003 — Ошибка при пустом поле Username**
- Предусловия: открыта страница логина
- Шаги:
  1) Оставить поле username пустым
  2) Ввести password: `secret_sauce`
  3) Нажать `Login`
- Ожидаемый результат: вход не выполнен, показано сообщение “Username is required” (или аналогичное).

**TC-LOGIN-004 — Ошибка при пустом поле Password**
- Предусловия: открыта страница логина
- Шаги:
  1) Ввести username: `standard_user`
  2) Оставить поле password пустым
  3) Нажать `Login`
- Ожидаемый результат: вход не выполнен, показано сообщение “Password is required” (или аналогичное).

**TC-LOGIN-005 — Заблокированный пользователь не может войти**
- Предусловия: открыта страница логина
- Шаги:
  1) Ввести username: `locked_out_user`
  2) Ввести password: `secret_sauce`
  3) Нажать `Login`
- Ожидаемый результат: вход не выполнен, показано сообщение о том, что пользователь заблокирован.

## Каталог (Products)

**TC-PROD-001 — Отображается список товаров после успешного входа**
- Предусловия: выполнен вход (см. TC-LOGIN-001)
- Шаги:
  1) Убедиться, что открыта страница `Products`
- Ожидаемый результат: отображаются карточки товаров (название, цена, кнопка Add to cart).

**TC-PROD-002 — Добавление товара в корзину из каталога**
- Предусловия: выполнен вход, открыта страница `Products`
- Шаги:
  1) Нажать `Add to cart` на любом товаре (например, первом в списке)
- Ожидаемый результат:
  - кнопка меняется на `Remove`
  - счётчик корзины (badge) показывает `1`.

**TC-PROD-003 — Удаление товара из корзины из каталога**
- Предусловия: выполнен вход, товар уже добавлен в корзину (badge = 1)
- Шаги:
  1) Нажать `Remove` у добавленного товара
- Ожидаемый результат:
  - кнопка меняется на `Add to cart`
  - счётчик корзины исчезает или становится `0` (в зависимости от реализации).

## Корзина (Cart)

**TC-CART-001 — Переход в корзину**
- Предусловия: выполнен вход
- Шаги:
  1) Нажать на иконку корзины в правом верхнем углу
- Ожидаемый результат: открылась страница `Your Cart`.

**TC-CART-002 — В корзине отображается добавленный товар**
- Предусловия:
  - выполнен вход
  - в каталоге добавлен 1 товар (см. TC-PROD-002)
- Шаги:
  1) Перейти в корзину
- Ожидаемый результат: в корзине отображается добавленный товар (название/цена/кол-во = 1).
