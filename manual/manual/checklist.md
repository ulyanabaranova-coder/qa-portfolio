# Checklist — SauceDemo (https://www.saucedemo.com)

## 1) Login page
- [ ] Страница логина открывается по https://www.saucedemo.com
- [ ] На странице есть поля Username и Password
- [ ] На странице есть кнопка Login
- [ ] Успешный логин с валидными данными (standard_user / secret_sauce)
- [ ] Ошибка при неверном пароле
- [ ] Ошибка при неверном логине
- [ ] Ошибка при пустом Username
- [ ] Ошибка при пустом Password
- [ ] Ошибка при пустых Username и Password
- [ ] Сообщение об ошибке видно и понятно пользователю
- [ ] После ошибки поля/страница ведут себя корректно (нет “поломки” интерфейса)
- [ ] Пользователь locked_out_user не может войти (сообщение о блокировке)

## 2) Products page (Catalog)
- [ ] После успешного входа открывается страница Products
- [ ] Отображается список товаров
- [ ] У товара есть название
- [ ] У товара есть цена
- [ ] У товара есть изображение
- [ ] У товара есть кнопка Add to cart
- [ ] Добавление товара меняет кнопку на Remove
- [ ] После добавления товара появляется badge корзины (счётчик)
- [ ] Можно добавить 2 разных товара — badge показывает 2
- [ ] Можно удалить товар (Remove) — badge уменьшается/исчезает
- [ ] Переход в карточку товара (если доступно) открывается корректно
- [ ] Сортировка товаров (если есть) переключается и меняет порядок списка

## 3) Cart (Your Cart)
- [ ] Переход в корзину по иконке корзины открывает страницу Your Cart
- [ ] Пустая корзина отображается корректно (нет ошибок/мусор��)
- [ ] В корзине отображается добавленный товар (название/цена)
- [ ] Количество товара отображается корректно
- [ ] Удаление товара из корзины работает
- [ ] Кнопка Continue Shopping возвращает в каталог
- [ ] Кнопка Checkout (если товар есть) доступна и открывает следующий шаг

## 4) Navigation / Menu
- [ ] Открывается боковое меню (hamburger menu)
- [ ] Кнопка Logout выводит пользователя из системы
- [ ] После Logout при открытии Products требуется логин (нет доступа без авторизации)
